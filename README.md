# 🧟‍♂️ [Minus Nine]

> 엘리베이터를 타고 층을 오르며, 수집한 자원으로 무기를 강화해 좀비 웨이브를 돌파하는 로그라이트 슈팅 게임입니다.  
> **[🎮 Steam 상점 페이지 바로가기](https://store.steampowered.com/app/4406640/Minus_Nine/)**

## 🛠 Tech Stack
- **Engine:** Unity 
- **API & Platform:** Steamworks API
- **Version Control:** GitHub

<br>

## 📌 Key Features (핵심 게임 플레이)
- **웨이브 & 디펜스:** 닫혀버린 엘리베이터 안팎에서 쏟아지는 좀비 웨이브를 막아내며 생존해야 합니다.
- **성장과 파밍 (로그라이트 요소):** 층을 이동하며 바이오 샘플을 수집하고, 무기 교체 및 스탯 강화를 통해 점진적으로 성장합니다.
- **도전과제 시스템:** '인류의 진화(첫 총 습득)', '지상으로..(클리어)' 등 플레이어의 도전 욕구를 자극하는 스팀 업적이 구현되어 있습니다.
